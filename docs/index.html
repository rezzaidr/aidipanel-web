FastCGI Cache

Native Nginx page caching with per-site controls, safe bypass rules, purge tools, and hit-rate monitoring.

🔒

Per-Site Isolation

Every site receives a dedicated Linux user and PHP-FPM pool. The panel runs separately as aidipanel.

🧰

Complete Site Toolkit

Manage files, databases, phpMyAdmin, cron, PHP settings, SSL, backups, SFTP, and access rules.

🐘

Multi-PHP

PHP 8.4 by default, with 8.2, 8.3, and 8.5 installed on demand and selected per site.

🛡️

Security Built In

UFW, Fail2ban, 2FA, role-based accounts, guarded deletion, and a narrow privileged wrapper.

⌨️

CLI + Web Panel

A comprehensive CLI plus a focused web interface for routine server and site operations.
